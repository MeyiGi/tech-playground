# Tech Playground

A personal repository where I explore different areas of software development, computer science, and software engineering.

This repository contains experiments, practice projects, algorithms, and examples built while studying topics like **Artificial Intelligence, Data Mining, Backend Development, Databases, Design Patterns, Messaging Systems, and more.**

Repository:  
https://github.com/MeyiGi/tech-playground

---

# Repository Structure

| Folder | Description |
|------|-------------|
| [100-pandas-problems](https://github.com/MeyiGi/tech-playground/tree/main/100-pandas-problems) | Solutions and practice exercises for mastering **Pandas** and data manipulation |
| [artificial_intelligence](https://github.com/MeyiGi/tech-playground/tree/main/artificial_intelligence) | AI experiments including agents, puzzles, search algorithms, and gradient descent |
| [assembler](https://github.com/MeyiGi/tech-playground/tree/main/assembler) | Low-level programming experiments and algorithms in **Assembly** |
| [clean-code](https://github.com/MeyiGi/tech-playground/tree/main/clean-code) | Examples demonstrating **clean architecture, MVC, and good code practices** |
| [data-minining](https://github.com/MeyiGi/tech-playground/tree/main/data-minining) | Data mining exercises including **data preprocessing, cleaning, and datasets** |
| [Database](https://github.com/MeyiGi/tech-playground/tree/main/Database) | SQL practice, database schemas, and relational database experiments |
| [Design Patterns](https://github.com/MeyiGi/tech-playground/tree/main/Design%20Patterns) | Implementations of classic **software design patterns in Java** |
| [docker](https://github.com/MeyiGi/tech-playground/tree/main/docker) | Experiments with **Docker** and containerized applications |
| [java-swing](https://github.com/MeyiGi/tech-playground/tree/main/java-swing) | GUI applications built using **Java Swing** |
| [OOP](https://github.com/MeyiGi/tech-playground/tree/main/OOP) | Object-Oriented Programming examples and small projects |
| [PyTest](https://github.com/MeyiGi/tech-playground/tree/main/PyTest) | Python testing examples including fixtures, mocks, parametrization, and class-based tests |
| [RabbitMQ-Patterns](https://github.com/MeyiGi/tech-playground/tree/main/RabbitMQ-Patterns) | Messaging patterns implemented with **RabbitMQ** |
| [scripting_language](https://github.com/MeyiGi/tech-playground/tree/main/scripting_language) | Various experiments with scripting languages (HTML, JS, PHP, etc.) |
